Помещайте сюда файлы шрифтов.

Используйте только форматы woff2 и woff2! Для преобразования шрифтов в указанные форматы можно использовать данный сервис: https://www.fontsquirrel.com/tools/webfont-generator

Не забудьте указать шрифты в файле index.html. Пример находится в том же файле в теге <style> под строкой "/*Fonts*/" (сам пример закомментирован).

Из-за технических особенностей Clickteam Fusion 2.5 во время выполнения приложения вам придётся повторно указать названия шрифтов с помощью действия "Указать шрифт" у объектов "Строка", "Поле ввода" и прочих. В качестве имени пишите то, что вы указали в качестве font-family в вашем index.html файле.

Файл _README.txt следует удалить.
